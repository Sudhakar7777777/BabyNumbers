Steps to launch your spring boot app in heroku:

Step#0: 
Initialize git and upload your code to a new project in Github or Gitlab

Step#1:
Download the heroku toolbelt
https://toolbelt.heroku.com/download/osx

Step#2:
Create a heroku account, setup SSH Keys

Step#3:
heroku create

Step#4:
git push heroku master

Step#5:
heroku open
